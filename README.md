## 1. 比赛重点难点剖析

- 比赛要求同时使用PaddleHub和WeChaty，有的小伙伴可能两个都不熟悉

- 项目本质上是使用WeChaty构建一个ChatBot

- 在此基础上接入PaddleHub可以使我们的ChatBot变得更智能

- 当然，最重要的是创意，创意，创意！！

## 2. 项目的产生背景

- 一般的机器人要么对所有好友开放，要么对指定好友开放

- 我想做一个可以随时控制是否对好友开放的机器人
